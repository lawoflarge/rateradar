# RateRadar Pricing

RateRadar is free.

- **iPhone app** — Free to download on the App Store. No account required.
- **Web app** — Free to use at https://rateradar-web.vercel.app.
- **In-app purchases** — None at this time.
- **Subscriptions** — None at this time.

RateRadar is supported by advertising and transparent broker affiliate partnerships, which cover its infrastructure costs. A Pro tier with custom alerts and CSV/API data export is planned for the future but is not yet available.

Not financial advice.
